# ETHIO Safeguard
## Fault-Tolerant Distributed System for Real-Time Aid Delivery Truck Tracking

### Team Members:
- GRAVEYARDOG: Backend & DevOps
- Tut-Gatwech-Tut: Frontend & Truck Simulator

### Architecture:
- Backend: Spring Boot with fault tolerance
- Frontend: React.js Admin Dashboard & Driver Interface
- Database: PostgreSQL with replication
- Cache: Redis for real-time data
- Load Balancer: Nginx
